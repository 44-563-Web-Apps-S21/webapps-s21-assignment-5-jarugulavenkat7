When the size of the browser is reduced, the row with plant names breaks first and all the plant names will stack on top of each other.

Explanation:
Since we have used grid class ".col-md" for the plant names the columns will collapse when the screen width is reduced less than 768px, because the screen width of medium devices is equal to or greater than 768px. Similarly, the row with letters will collapse when the screen width is reduced less than 576px.
